# 🎓 Learnbit - Learning Rewards Protocol

> Incentivize study hours with tokens on the Stacks blockchain! 📚

## 🌟 Overview

Learnbit is a decentralized protocol that rewards students for their study time with fungible tokens. Track your study hours, earn rewards, and build a verifiable learning history on-chain.

## ✨ Features

- 🔐 Student registration system
- ⏱️ Study session tracking
- 💰 Automatic token rewards
- 🏆 Study streak tracking
- ✅ Verification system
- 💸 Token transfers between students

## 🚀 Getting Started

1. Deploy the contract using Clarinet
2. Register as a student using `register-student`
3. Start tracking study time with `start-study-session`
4. End your session and claim rewards with `end-study-session`

## 📋 Main Functions

- `register-student`: Create your student profile
- `start-study-session`: Begin tracking study time
- `end-study-session`: Complete session and earn rewards
- `transfer-rewards`: Send tokens to other students
- `get-student-info`: View student statistics
- `get-balance`: Check token balance

## ⚙️ Configuration

- Default reward rate: 10 tokens per hour
- Minimum study time: 30 minutes
- Cooldown period: 24 hours

## 🔒 Security

- Owner-only administrative functions
- Verification system for trusted students
- Built-in cooldown mechanisms
